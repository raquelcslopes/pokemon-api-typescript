## Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn

### Installation

```bash
npm install
# or
yarn install
```

### Running the App

```bash
npm run dev
# or
yarn dev
```

The app will be available at [http://localhost:5175](http://localhost:5175) by default.

### Running Tests

```bash
npx playwright test
# or
yarn playwright test
```

### Building for Production

```bash
npm run build
# or
yarn build
```
